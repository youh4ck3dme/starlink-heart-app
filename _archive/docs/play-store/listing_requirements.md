# Google Play Store Assets Checklist

## 1. App Icon (store-icon.png)
- [x] Resolution: 512px x 512px
- [x] Format: 32-bit PNG
- [x] Transparency: **NONE** (Solid background)
- [x] Max Size: 1024KB
- [x] Content: No "New" or "Best" badges.

## 2. Feature Graphic (feature-graphic.png)
- [x] Resolution: 1024px x 500px
- [x] Format: JPEG or PNG
- [x] Transparency: **NONE**
- [x] Content: Branding integrated, no cropping of critical elements.

## 3. Screenshots (Phone)
- [ ] Required: Min 2, Aspect Ratio 9:16 (Portrait)
- [ ] Max size: 8MB per image
- [ ] Placeholders: You can take screenshots of the running app using `npm run dev` and your browser's dev tools (Cmd+Shift+4).

## 4. Screenshots (Tablet)
- [ ] Required for "Designed for Tablets" category.
- [ ] 7-inch & 10-inch variants.

## Located at:
All assets are prepared in this folder (`docs/play-store/`).
- `feature-graphic.png`
- `store-icon.png`
